# In-the-Long-Run

# In the Long Run — Game Design Document

**Version**: 1.0（統合版）

---

## 1. コンセプト

> **「なぜ国家は繁栄し、なぜ衰退するのか」を、プレイヤー自身の意思決定を通じて体感するゲーム**

公共選択論とマクロ経済学を中核メカニクスに据え、プレイヤーは国家の政治・経済・外交を導く。歴史的な制度変遷を組み込み、古代から現代まで、制度の選択と進化が国家の命運を分ける。

タイトル「In the Long Run」は、ケインズの名言「In the long run we are all dead（長期的にはわれわれは皆死んでいる）」に由来する。あらゆる局面でプレイヤーは「長期的には正しいが、短期的には選べない」というジレンマに直面する。

**三本柱:**
- **公共選択論** — 政治家・官僚・有権者・利益集団の合理的行動が、集団として不合理な結果を生む構造
- **現代マクロ経済学** — ニューケインジアン経済学と行動経済学の知見を反映した経済モデル
- **歴史的制度変遷** — 制度は固定ではなく、条件に応じて進化し、組み合わせの相性が国家の効率を左右する

---

## 2. ゲームモード

### 2.1 一国操作モード（キャンペーンモード）

プレイヤーはひとつの国家を選択し、その興亡を体験する。

- **時間軸**: 1ターン＝1年。数百年スパンのプレイが可能
- **開始時代**: 任意の時代から開始可能（古代〜現代）
- **勝利条件**: 明確な「勝ち」はなく、国家の存続年数・最大繁栄度・歴史的評価がスコアとなる
- **敗北条件**: 財政破綻、革命による政権崩壊、外国による併合、国家分裂
- **難易度**: 初期条件（地理・資源・周辺国・開始時代）によって自然に決まる
- **シナリオ例**:
  - 資源豊富な新興国（資源の呪いを回避できるか？）
  - 老朽化した大国（制度疲労をどう立て直すか？）
  - 小国サバイバル（大国間の緩衝地帯で生き残れるか？）
  - 古代文明の興亡（マルサスの罠を脱出できるか？）
  - 近代化の岐路（西洋的近代化 vs 独自路線？）
  - 体制移行の苦悩（計画経済崩壊後の再建）

### 2.2 サンドボックスモード

世界全体を俯瞰し、複数国家の相互作用を観察・操作する。

- **世界生成**: 大陸・資源・民族分布をランダムまたはカスタム生成
- **介入レベル**: 任意の国に随時「乗り移り」可能。放置して観察もできる
- **実験用途**: 「全国家を民主主義にしたら？」「全国家が重商主義を採用したら？」
- **観戦モード**: 全自動で世界史を生成し、統計グラフで眺める
- **パラメータ調整**: 経済モデルの係数・政治的傾向・行動経済学パラメータを自由に調整
- **比較分析**: 複数のシミュレーション結果を並列比較
- **AIの行動ログ**: 各国のAIがなぜその政策を選択したかの理由を表示
- **経済学者モード**: 特定の経済学派の理論に基づく政策セットを試す

## 3. 歴史的制度変遷システム

### 3.1 概要

制度は固定された選択肢ではなく、**経済・技術・社会の条件によって解禁（unlock）され、政治的意思決定によって採用される**動的なもの。各国家ごとに独立して進行し、ある国が産業革命を迎えていても別の国はまだ封建制にとどまる非対称性が重要。

制度変遷は自動的には起こらない。条件が整っても、利益集団の抵抗・プレイヤーの選択・偶発的イベントにより、変遷は加速も停滞もする。

> 💡 **Tip**: *「歴史は制度の産物であり、制度は歴史の産物である」* — ダグラス・ノース

### 3.2 時代区分

| 時代 | 典型的年代 | 経済基盤 | 支配的制度 | 中核的課題 |
|------|-----------|---------|----------|-----------|
| **古代・部族社会** | 〜500頃 | 狩猟採集・初期農業 | 部族合議・首長制 | 生存・定住・余剰の創出 |
| **古典古代** | 500 BC〜500 | 農業・奴隷制 | 都市国家・帝国 | 領域支配・法の統一・交易網 |
| **封建時代** | 500〜1400 | 荘園農業 | 封建制・教会権力 | 土地支配・領主間競争 |
| **初期近代** | 1400〜1700 | 商業資本主義 | 絶対王政・重商主義 | 中央集権化・大航海・宗教改革 |
| **啓蒙と革命** | 1700〜1850 | 初期工業化 | 立憲制・共和制の萌芽 | 人権思想・市民革命・産業革命 |
| **帝国主義時代** | 1850〜1914 | 工業資本主義 | 国民国家・帝国主義 | 植民地獲得・大国間競争・社会問題 |
| **世界大戦期** | 1914〜1945 | 総力戦経済 | 全体主義の台頭 | 総力戦・イデオロギー対立・恐慌 |
| **冷戦期** | 1945〜1991 | 混合経済 | 福祉国家 vs 計画経済 | 核抑止・脱植民地化・開発 |
| **グローバル化時代** | 1991〜2020 | グローバル資本主義 | 自由民主主義・新自由主義 | グローバル化・金融危機・格差 |
| **現代** | 2020〜 | デジタル経済 | 権威主義の復権？ | AI・気候変動・分断・監視 |

### 3.3 時代ごとの経済モデル適用

時代によって利用可能な経済モデルと政策手段が変化する。

| 時代 | 適用される経済モデル | 利用可能な政策 |
|------|-------------------|-------------|
| 古代〜封建 | マルサスモデル（人口と生産の関係） | 現物徴税、公共事業（灌漑）、戦争 |
| 初期近代 | 重商主義モデル（貿易差額） | 関税、特許、植民地経営 |
| 産業革命期 | 古典派モデル（セイの法則的世界） | 自由貿易、金本位制、最小国家 |
| 20世紀以降 | ニューケインジアンモデル（完全版） | 財政政策、金融政策、福祉政策、行動政策 |

### 3.4 文明の転換点（Critical Junctures）

制度が大きく変わりうる分岐点。特別なイベントとして発生する。

| 転換点 | 歴史的例 | ゲーム内効果 |
|--------|---------|------------|
| **疫病** | 黒死病 | 人口急減→労働力希少化→農奴解放圧力。封建制の動揺 |
| **軍事技術革新** | 火器の普及 | 騎士の没落→常備軍→中央集権化→課税能力の向上 |
| **新大陸/交易路変化** | 大航海時代 | 商業革命→重商主義→植民地帝国 |
| **宗教改革** | ルターの95か条 | 教会権力の動揺→世俗権力強化→識字率向上 |
| **市民革命** | フランス革命 | 旧体制崩壊→法的平等→ナショナリズム |
| **産業革命** | イギリス18世紀末 | マルサスの罠からの脱出→都市化→民主化圧力 |
| **大恐慌** | 1929年 | 自由放任の限界露呈→ケインズ主義→福祉国家 |
| **大戦争** | 世界大戦 | 帝国崩壊→民族自決→国際秩序再編 |
| **体制崩壊** | ソ連崩壊 | 計画経済の終焉→移行経済の困難 |
| **金融危機** | 2008年 | 新自由主義への信頼喪失→ポピュリズム台頭 |
| **パンデミック** | 各時代の疫病 | 国家能力の試金石。社会変革の触媒 |

> 💡 **Tip**: *「重大な転換点において、小さな制度的差異が、数世紀にわたる発展経路の大きな分岐を生む」* — ダロン・アセモグル＆ジェイムズ・A・ロビンソン

### 3.5 王朝循環モデル（帝国型国家向け）

```
[建国期] 有徳の創業者。制度の刷新。活力ある政治
  ↓ (1〜3世代)
[盛期] 制度の成熟。経済繁栄。文化の開花。領土の最大化
  ↓ (2〜4世代)
[停滞期] 宮廷の奢侈。官僚の腐敗。利益集団の固着
  ↓ (1〜3世代)
[衰退期] 財政危機。地方の反乱。外敵の侵入
  ↓
[崩壊・内戦] → [新王朝の建国] → サイクルの繰り返し
```

> 💡 **Tip**: *「文明は三世代で衰退する。第一世代が築き、第二世代が維持し、第三世代が浪費する」* — イブン・ハルドゥーン『歴史序説』

## 4. 制度の分類と進化

### 4.1 政治制度

#### 統治形態の進化ツリー

```
部族合議制
  ├→ 首長制
  │    ├→ 王制（世襲）
  │    │    ├→ 封建王制（分権的）
  │    │    │    ├→ 身分制議会
  │    │    │    │    ├→ 立憲君主制 → 議会制民主主義
  │    │    │    │    └→ 絶対王政
  │    │    │    │         ├→ 啓蒙専制 → 立憲制 or 革命
  │    │    │    │         └→ 市民革命 → 共和制 or 軍事独裁
  │    │    │    └→ 中央集権失敗 → 分裂・割拠
  │    │    └→ 帝国（多民族統治）
  │    │         ├→ 官僚帝国（中華型）→ 王朝循環
  │    │         └→ 征服帝国（モンゴル型）→ 分裂
  │    └→ 神権制 → 世俗化 or 停滞
  └→ 都市国家 → 寡頭制 / 民主制 / 僭主制
```

#### 政体の特徴

| 政体 | 特徴 | 公共選択論的含意 |
|------|------|----------------|
| **絶対君主制** | 意思決定が速い。一貫性が高い | 独裁者のジレンマ：正確な情報が上がってこない |
| **立憲君主制** | 君主の権威＋議会の牽制 | 安定性は高いが改革が遅い |
| **民主共和制** | 選挙サイクルが政策を左右 | 政治的景気循環、合理的無知、中位投票者定理 |
| **一党独裁制** | 長期計画が可能。腐敗が蓄積 | 党内派閥のレントシーキング |
| **軍事政権** | 治安安定。経済自由度低い | 軍産複合体の制度化 |
| **神権政治** | イデオロギー的結束が強い | 聖職者階級のレントシーキング |

> 💡 **Tip**: *「民主主義は最悪の政治形態である——これまでに試みられてきた他のあらゆる形態を除けば」* — ウィンストン・チャーチル

#### 政治制度の解禁条件（例）

| 制度 | 前提条件 | 加速要因 | 阻害要因 |
|------|---------|---------|---------|
| **絶対王政** | 常備軍、官僚機構、商業税収 | 封建貴族の弱体化 | 貴族の団結、議会の伝統 |
| **立憲制** | 識字率>20%、商業ブルジョワジー | 財政危機、啓蒙思想 | 絶対王政の安定 |
| **議会制民主主義** | 識字率>50%、中産階級の形成 | 都市化、メディア | 農村社会の温存 |
| **普通選挙** | 議会制の確立、労働運動 | 総力戦（兵士→参政権） | 保守エリートの反対 |

#### 参政権拡大の力学

- **拡大圧力**: 大衆動員（戦争）、都市化、識字率向上、経済危機、外国の民主化
- **拡大への抵抗**: 既得有権者の政治的希薄化への恐怖、エリートの再分配恐怖
- **公共選択論的分析**: エリートが民主化を受け入れるのは、革命の脅威が参政権拡大のコストを上回ったとき

> 💡 **Tip**: *「エリートが民主化を受け入れるのは、善意からではなく、革命の脅威が十分に深刻だからである」* — アセモグル＆ロビンソン

### 4.2 経済体制

#### 経済体制の進化ツリー

```
自給自足経済
  ├→ 貢納経済
  │    ├→ 荘園経済（封建的）
  │    │    ├→ 商業経済 → 重商主義
  │    │    │    ├→ 自由市場経済
  │    │    │    │    ├→ 自由放任資本主義
  │    │    │    │    │    ├→ 社会的市場経済 → 福祉国家型混合経済
  │    │    │    │    │    │                  └→ 新自由主義
  │    │    │    │    │    └→ 社会主義革命 → 計画経済
  │    │    │    │    │         ├→ 市場社会主義 / 体制崩壊
  │    │    │    │    │         └→ 国家資本主義
  │    │    │    │    └→ 独占資本主義 → 独占禁止法
  │    │    │    └→ 国家主導型発展（開発独裁）
  │    │    └→ 農奴制の強化（東欧型）→ 工業化の遅延
  │    └→ 官僚制的再分配（帝国型）
  └→ 交易経済（商業都市国家型）
```

#### 各経済体制の要約

| 経済体制 | GDP構成 | 政策手段 | 利益集団 | 公共選択論的特徴 |
|---------|---------|---------|---------|----------------|
| **自給自足** | 農業90%+ | 現物徴収のみ | 部族長・宗教指導者 | マルサスの罠 |
| **荘園経済** | 農業70〜85% | 現物租・賦役 | 封建貴族・教会・ギルド | 貴族は中央集権に抵抗 |
| **重商主義** | 農業50〜60% | 関税・特許・植民地 | 王室・特許商人・軍部 | 特許商人と王権の癒着 |
| **自由放任** | 工業30〜50% | 最小国家・金本位制 | 産業資本家（支配的） | 資本家が政治を支配 |
| **計画経済** | 重工業偏重 | 五カ年計画・価格統制 | 党幹部・計画官僚 | ニスカネンモデルの極致 |
| **混合経済** | サービス50〜70% | ケインズ的財政・社会保険 | 労組・資本家・年金受給者 | 福祉の不可逆性 |
| **新自由主義** | 金融・サービス60〜80% | 規制緩和・民営化 | 金融セクター・多国籍企業 | 規制緩和がレントシーキングの対象に |
| **国家資本主義** | 国営企業30〜60% | 国営企業・産業政策 | 党幹部兼経営者 | 政治と経済の癒着の制度化 |

> 💡 **Tip**: *「社会主義計画経済の根本問題は、分散した知識を中央に集約できないことである」* — フリードリヒ・ハイエク

### 4.3 金融制度

```
物々交換 → 商品貨幣 → 鋳造貨幣 → 金本位制 → 管理通貨制度 → 金融自由化
                                         │
                                    信用の発展
                              高利貸し → 銀行 → 中央銀行 → 株式市場 → デリバティブ
```

| 金融制度 | 利用可能な政策 | リスク |
|---------|-------------|-------|
| 物々交換 | 金融政策なし | 取引コスト極大 |
| 商品貨幣 | 限定的な課税 | 貨幣量が鉱山依存 |
| 鋳造貨幣 | 改鋳による「インフレ税」 | 通貨の信認低下 |
| 金本位制 | 安定通貨。国際貿易促進 | デフレ圧力。金融政策の柔軟性なし |
| 管理通貨制度 | 完全なマクロ経済政策が可能 | ハイパーインフレ。信認が鍵 |
| 金融自由化 | 資本の効率的配分 | バブル。システミックリスク |

> 💡 **Tip**: *「金は野蛮の遺物である」* — ケインズ。金本位制は安定をもたらすが、不況時に金融政策の手を縛る。

### 4.4 法制度

```
慣習法 → 王の裁判 → 成文法
              ├→ ローマ法継受（大陸法系）→ 法の支配
              ├→ コモン・ロー（判例法系）→ 法の支配
              └→ 宗教法（シャリーア、カノン法）
```

| 法制度の要素 | 経済への影響 | 重要度 |
|------------|------------|-------|
| 財産権の保護 | 投資インセンティブ | 極めて重要 |
| 契約の執行 | 取引コスト低下 | 極めて重要 |
| 会社法 | 株式会社・有限責任 | 重要 |
| 破産法 | 再挑戦の可能性 | 重要 |
| 独占禁止法 | 競争の維持 | 重要 |

> 💡 **Tip**: *「経済発展の鍵は効率的な財産権にある」* — ダグラス・ノース

### 4.5 軍事制度

```
部族戦士 → 貴族私兵 → 傭兵軍 → 常備軍 → 徴兵制（国民軍）→ 志願制 → 民間軍事会社
```

| 軍事技術 | 社会的影響 | 制度への影響 |
|---------|----------|------------|
| 重装歩兵 | 市民の軍事的重要性 | 民主制の基盤（ギリシア） |
| 騎兵と城 | 騎士階級の支配 | 封建制の基盤 |
| 火器 | 騎士の没落 | 中央集権国家の台頭 |
| 国民軍 | 全国民の戦争参加 | 参政権拡大。ナショナリズム |
| 核兵器 | 大国間直接戦争の抑止 | 代理戦争。冷戦構造 |

### 4.6 社会制度

**身分制度:** 未分化社会 → カースト制/奴隷制/封建身分制 → 市民革命による法的平等 → 経済的階級社会

**教育制度:** 宗教教育独占 → 印刷革命 → 初等教育義務化 → 中等教育普及 → 高等教育大衆化 → 生涯学習社会

## 5. 制度の導入と相性

### 5.1 二つの導入経路

#### 自然導入（Organic Adoption）

社会的・経済的条件が80%以上充足すると、制度が選択肢として出現する。移行コストが低く、定着率が高い。

#### 強制導入（Forced Adoption）

前提条件が30〜79%の段階で、国家権力を行使して制度を導入する。

**必要な権力資源:**
- **軍事力** — 反対勢力の物理的鎮圧能力
- **政治的権威** — 指導者・政権の正当性と求心力
- **財政力** — 制度移行コストを賄う経済力
- **官僚の実行能力** — 新制度を現場で運用する行政能力
- **イデオロギー的正当化** — 「なぜ変える必要があるか」の説得的物語

**権力算出式:**
```
有効権力 = 軍事力 × 政治的権威 × min(財政力, 官僚能力) × (1 + イデオロギー補正)
導入必要権力 = 基礎コスト × (1 - 前提条件充足度) × 制度間不適合度 × 抵抗勢力係数
```

> 💡 **Tip**: *「改革者の最大の敵は、旧制度から利益を得る者すべてと、新制度から利益を得るか確信できない者すべてである」* — マキャヴェッリ『君主論』

### 5.2 歪み（Distortion）の蓄積

強制導入された制度は「歪み値」を持つ。

```
歪み値 = (1 - 前提条件充足度) × 基礎歪み + Σ(制度間不適合ペナルティ)
```

| 歪み値 | 状態 | 効果 |
|-------|------|------|
| 0〜0.2 | 軽微な摩擦 | 制度効率 -10〜20%。自然解消の可能性 |
| 0.2〜0.4 | 顕著な歪み | 効率 -20〜40%。闇経済の成長 |
| 0.4〜0.6 | 深刻な矛盾 | 効率 -40〜60%。社会不安 |
| 0.6〜0.8 | 危機的状態 | ほぼ機能しない。反乱リスク急上昇 |
| 0.8〜1.0 | 崩壊寸前 | 制度の自壊。革命・内戦の可能性 |

**歪みの減衰要因:** 時間経過（世代交代: 20〜30ターン）、経済成長、教育、制度の追加改革、国際的信認の獲得

> 💡 **Tip**: *「新しい思想が勝利するのは、反対者がやがて死に、新思想に馴染んだ世代が成長するからである」* — マックス・プランク

### 5.4 強制導入の典型ケース: 絶対王政 × 管理通貨制度

管理通貨制度は中央銀行の独立性・議会の財政監視・透明な統計を前提とし、絶対王政とは以下の点で致命的に不適合：

| 管理通貨制度が要求するもの | 絶対王政が提供するもの | 不適合度 |
|------------------------|--------------------|----|
| 中央銀行の独立性 | 君主が全権を持つ | 致命的 |
| 財政規律のメカニズム | 君主の支出を制限する制度がない | 致命的 |
| 通貨への信認 | 個人の信認に依存（脆弱） | 重大 |
| 透明な経済統計 | 正確な情報が上がらない | 重大 |

**予想展開:** 短期的に通貨発行で経済活性化 → 財政規律の欠如 → 通貨過剰発行 → インフレ → ハイパーインフレ or 立憲制への移行圧力

**核心的ジレンマ:** 管理通貨の利点（財政の柔軟性）は、絶対王政のリスク（規律の欠如）と表裏一体。恩恵を受けつつリスクを管理するには、自分の権力を制限する必要がある。

> 💡 **Tip**: *「紙幣を発行する権力は、紙幣を濫発する誘惑を伴う」* — ミルトン・フリードマン

### 5.5 歴史的に成功した制度パッケージ

| パッケージ名 | 構成要素 | ボーナス | 歴史的モデル |
|-------------|---------|---------|------------|
| **ウェストミンスター型** | 立憲君主制 + コモン・ロー + 自由市場 + 管理通貨 | 制度の質+25% | イギリス、カナダ |
| **北欧モデル** | 議会制民主主義 + 混合経済 + 高福祉 | 制度の質+20%、ジニ改善+30% | スウェーデン |
| **開発独裁型** | 一党独裁 + 国家資本主義 + 産業政策 | GDP成長+2%、腐敗蓄積+50% | 韓国、シンガポール |
| **ライン型** | 議会制 + 社会的市場経済 + 協調的労使関係 | 製造業効率+20% | ドイツ |
| **中華帝国型** | 官僚帝国 + 科挙 + 成文法 | 官僚の質+30%、技術革新-20% | 唐〜清 |

### 5.6 不安定な制度パッケージ

| パッケージ名 | 内在する矛盾 | 典型的結末 |
|-------------|------------|-----------|
| **ファシスト型** | 永続的拡張を必要とする | 戦争→敗北→崩壊 |
| **レンティア国家型** | 資源価格に従属。制度構築のインセンティブ欠如 | 資源枯渇→財政危機 |
| **ポピュリスト型** | 短期人気と長期持続性の致命的矛盾 | インフレ→通貨危機→IMF介入 |

### 5.7 改革の順序問題（Sequencing）

推奨される一般的順序:
```
1. 法の支配の確立
2. 基本的な行政能力
3. 経済の自由化
4. 金融制度の近代化
5. 政治的自由化
6. 社会保障の整備
```

順序を間違えた典型的失敗: 法の支配なき市場化→オリガルヒ支配、行政能力なき民主化→部族対立激化、金融自由化が先行→バブルと金融危機

> 💡 **Tip**: *「民主主義、国家能力、法の支配——この三つが揃ってはじめて政治的秩序が成立する」* — フランシス・フクヤマ『政治の起源』

### 5.8 経路依存性

一度確立された制度は変更コストが高く、非効率でも存続する。初期の制度選択が後の選択肢を制約する。

- 「封建制→絶対王政→立憲制」ルートと「帝国→官僚制→共産主義」ルートでは、「民主化」の難易度が全く異なる
- 旧制度の遺産（慣行・文化・利益集団）が新制度下でも影響力を持つ

> 💡 **Tip**: *「制度は粘着性を持つ。非効率であっても、変更コストの高さゆえに存続する」* — ダグラス・ノース

## 6. 政治システム（公共選択論）

### 6.1 利益集団とロビイング

**主要な利益集団:**

| 利益集団 | 主な要求 | 対立する相手 |
|---------|---------|------------|
| 地主・農業ロビー | 農業補助金・関税保護 | 消費者・自由貿易派 |
| 産業資本家 | 低税率・規制緩和 | 労働組合・環境団体 |
| 労働組合 | 最低賃金引上げ・社会保障拡充 | 産業資本家 |
| 軍部 | 国防費増額・対外強硬策 | 福祉ロビー |
| 官僚機構 | 予算の自己増殖 | 改革派 |
| 宗教団体 | 教育への影響力・道徳的規制 | 世俗主義者 |
| 金融セクター | 金融自由化・低規制 | 規制派 |

**メカニクス:**
- 各集団は「影響力ポイント」を持ち、政策決定に圧力をかける
- 長期的に特定集団が強くなりすぎると「制度の硬直化」（オルソンの理論）
- 影響力が閾値を超えると、その集団に不利な改革は政治的に不可能になる（ロックイン効果）

> 💡 **Tip**: *「集中した利益は、拡散した費用に常に勝つ」* — マンカー・オルソン

### 6.2 官僚機構モデル

官僚はプレイヤーの直接制御下にない**自律的エージェント**。ニスカネンの予算最大化モデルをベースとする。

> 💡 **Tip**: *「官僚は公益の奉仕者ではない。自らの予算と権限を最大化する合理的経済人である」* — ウィリアム・ニスカネン

#### 官僚の行動原理

```
官僚の効用 = f(予算規模, 権限範囲, 裁量の自由度, 組織の存続)
```

#### 組織命題の歪み（Mission Distortion）

組織は本来の目的を忘れ、自己保存と拡大が目的化しうる。

| 歪みの類型 | 説明 | 国家への害 |
|-----------|------|-----------|
| **使命肥大化** | 管轄を拡大解釈 | 他省庁との重複、予算浪費 |
| **使命代替** | 本来の使命を放棄し別のことをする | 目的が達成されないまま予算だけ消費 |
| **使命硬直化** | 環境変化に適応できない | 技術革新への対応不全 |
| **自己言及化** | 組織の維持自体が使命に | 無駄な規制の増殖 |
| **捕獲（Regulatory Capture）** | 規制対象業界に取り込まれる | 規制の形骸化 |

> 💡 **Tip**: *「測定が目標になると、それは良い測定ではなくなる」* — グッドハートの法則

#### 予算削減への官僚の反応

| 削減幅 | 反応 | 効果 |
|--------|------|------|
| 微減（〜5%） | 渋々受容 | 実質的影響なし |
| 中程度（5〜15%） | ワシントン記念碑作戦 | 最も人気のあるサービスを選択的に削減して世論を誘導 |
| 大幅（15〜30%） | 組織的サボタージュ | 政策実施率の低下。改革の骨抜き |
| 壊滅的（30%超） | 組織崩壊リスク | 行政機能の麻痺。ただし構造改革のチャンス |

#### 官僚機構の長期ダイナミクス

```
健全な組織 → 肥大化 → 組織命題の歪み → 制度的硬直化 → 【分岐】改革成功 or さらなる硬直化
```

**鍵となる洞察:** 官僚機構は放置すると必ず肥大化・硬直化する。防ぐには継続的な政治的コストが必要だが、政治家にそのインセンティブは弱い。

### 6.3 選挙と世論

- **中位投票者定理**: 政策は中間層に引き寄せられる
- **合理的無知**: 有権者は詳細な政策を理解するインセンティブが薄い
- **情報コスト**: メディアの発達度が有権者の情報水準を左右
- **投票率**: 経済状況・分極化・制度（義務投票など）に依存

> 💡 **Tip**: *「合理的な有権者は政治について無知であることが合理的である」* — アンソニー・ダウンズ

### 6.4 腐敗と衰退

**腐敗度の影響要因:** 権力の集中度、官僚の給与水準、司法の独立性、メディアの自由度、政権の長期化、資源収入依存度

**腐敗の効果:** 税収漏洩、インフラ投資の非効率化、外国投資の忌避、頭脳流出、軍の戦闘力低下

```
利益集団の固定化 → 制度の硬直化 → 改革の不可能性 → 経済停滞 → 不満 → 体制危機
```

> 💡 **Tip**: *「国家が失敗するのは、収奪的制度が包括的制度に取って代わるときである」* — アセモグル＆ロビンソン『国家はなぜ衰退するのか』

## 7. 経済システム（現代マクロ経済学）

### 7.1 ニューケインジアン経済モデル

古典派との違いをプレイヤーが体感できる：
- 価格と賃金は即座に調整されない（硬直性）
- 不況時に「市場に任せれば回復する」わけではない
- 金融政策は短期的には実体経済に影響する
- 期待（expectations）が経済を大きく左右する

> 💡 **Tip**: *「長期的にはわれわれは皆死んでいる」* — ケインズ。短期の価格硬直性と需要不足は現実の問題である。

#### ニューケインジアン・フィリップス曲線

```
π_t = β × E_t[π_(t+1)] + κ × ỹ_t + ε_t
```

- **π_t**: 今期のインフレ率
- **E_t[π_(t+1)]**: 来期のインフレ期待
- **ỹ_t**: GDPギャップ
- **κ**: 価格硬直性の度合い
- **ε_t**: 供給ショック

中央銀行の**信認（credibility）**がインフレ期待を安定させる鍵。信認が崩壊すると引き戻すのに大きなコスト。

#### IS曲線

```
ỹ_t = E_t[ỹ_(t+1)] - σ × (i_t - E_t[π_(t+1)] - r*) + g_t
```

#### テイラー・ルール

```
i_t = r* + π* + α_π(π_t - π*) + α_y × ỹ_t
```

中央銀行の独立性が高い → テイラー・ルール的運営 → インフレ安定だが政治的不自由。

> 💡 **Tip**: *「金融政策のルールは裁量に勝る」* — ジョン・テイラー。だが政治家は常に「今回だけは特別」と裁量を求める。

### 7.2 基本指標

| 指標 | 説明 |
|------|------|
| GDP / GDP成長率 | 国力の基盤。前年比の成長 |
| 一人当たりGDP | 生活水準 |
| 潜在GDP / GDPギャップ | 完全雇用GDP。負なら不況 |
| インフレ率 | 物価上昇率 |
| 失業率 / NAIRU | 非自発的失業。インフレを加速させない失業率 |
| 国債残高/GDP比 | 財政の持続可能性 |
| ジニ係数 | 所得格差 |
| 貿易収支 | 輸出 - 輸入 |

### 7.3 生産関数

**長期（潜在GDP）:**
```
Y* = A × K^α × L^(1-α) × I
```

**短期（実際のGDP）:**
```
Y = Y* + GDPギャップ
```

- **A**: 全要素生産性、**K**: 資本ストック、**L**: 有効労働力、**I**: 制度の質（0〜1）

**制度の質 I:** 財産権保護、契約執行、腐敗の少なさ、規制の合理性、官僚の効率性、政治的安定性

### 7.4 財政政策

**歳入:**

| 税種 | 効果 | 政治的含意 |
|------|------|-----------|
| 所得税（累進） | 再分配効果大。高税率は労働意欲を削ぐ | 富裕層ロビーが反対 |
| 法人税 | 高すぎると投資逃避 | 資本家ロビーが減税要求 |
| 消費税 | 逆進的。徴収容易 | 低所得層に不人気 |
| 関税 | 国内産業保護 | 衰退産業ロビーが要求 |
| ピグー税 | 外部性の内部化 | 汚染産業の反対 |
| 土地税 | 経済学的に最も効率的 | 政治的に最も困難 |

**歳出:** 国防費、教育費、インフラ投資、社会保障費、研究開発費、官僚機構の人件費、利払い費

**財政乗数（ニューケインジアン的）:** 不況時に高い（1.5〜2.0）、好況時に低い（0.5〜1.0）、ゼロ金利下では特に有効

> 💡 **Tip**: *「好況期に節約し、不況期に支出せよ」* — ケインズ的処方箋。だが政治家は好況期に節約するインセンティブを持たない。

### 7.5 金融政策

- **金利操作**: 低金利→投資促進・インフレ / 高金利→インフレ抑制・景気抑制
- **中央銀行の独立性**: 低いと政治的景気循環が激化
- **ゼロ金利制約**: 深刻な不況で通常の金融政策が無効化 → 量的緩和、フォワードガイダンス
- **国際金融のトリレンマ**: 「固定相場」「自由な資本移動」「独立した金融政策」のうち同時に2つしか選べない

> 💡 **Tip**: *「流動性の罠においては、金融政策は紐を押すようなものだ」* — ポール・クルーグマン

### 7.6 景気循環

- 好況→過熱→不況→回復のサイクルが内生的に発生
- **政治的景気循環**: 選挙前に拡張→選挙後に引き締め（ノードハウスモデル）
- **自動安定化装置**: 累進所得税、失業保険が政治的意思決定なしに景気を安定化

### 7.7 経済危機イベント

| 危機 | 発生条件 | 効果 |
|------|---------|------|
| 財政危機 | 国債残高/GDP比が閾値超過 | 国債金利急騰→デフォルトリスク |
| 通貨危機 | 外貨準備不足＋固定相場制 | 通貨暴落 |
| バブル崩壊 | 過剰信用膨張 | 資産価格急落→金融危機 |
| ハイパーインフレ | 財政ファイナンスの常態化 | 通貨の信認崩壊 |
| スタグフレーション | 供給ショック＋金融緩和 | 不況とインフレの同時進行 |
| バランスシート不況 | 資産価格崩壊→債務超過 | 長期停滞 |

> 💡 **Tip**: *「安定は不安定化をもたらす」* — ハイマン・ミンスキー

## 8. 行動経済学の統合

### 8.1 有権者・国民の行動バイアス

| バイアス | ゲーム内の効果 |
|---------|-------------|
| **現在バイアス** | 年金改革・環境政策など長期的政策が不人気 |
| **損失回避** | 既得権の廃止は利益の創出より強い反対を生む |
| **現状維持バイアス** | 制度改革への抵抗が合理的水準を超える |
| **利用可能性ヒューリスティック** | 直近の大事件が政策の優先順位を歪める |
| **フレーミング効果** | 「増税」より「社会貢献金」の方が受容されやすい |
| **群衆行動** | 銀行取付、バブルの形成と崩壊 |
| **双曲割引** | 国債の過剰発行が「合理的に見えてしまう」 |

### 8.2 政策立案者のバイアス

- **計画の誤謬**: 大規模プロジェクトのコスト・期間を過小評価
- **サンクコストの誤謬**: 失敗プロジェクトにさらに投資
- **自信過剰バイアス**: 経済予測を過信した政策決定

> 💡 **Tip**: *「人間は合理的なのではない。合理化する生き物なのだ」* — ダニエル・カーネマン

### 8.3 ナッジと行動政策

| ナッジ政策 | コスト | 効果 |
|-----------|-------|------|
| デフォルト設定変更 | 極低 | 年金加入率・臓器提供率の向上 |
| 情報開示義務 | 低 | 市場透明性の向上 |
| 社会規範の活用 | 低 | 「隣人はこうしている」で納税率向上 |
| コミットメント装置 | 低 | 財政ルール・インフレ目標の法制化 |

> 💡 **Tip**: *「良い選択アーキテクチャは、人々の自由を奪わずに、より良い意思決定を促す」* — セイラー＆サンスティーン『ナッジ』

### 8.4 アニマル・スピリッツと期待

- 投資は「気分」に左右される。消費者信頼感指数が消費行動を変える
- **自己実現的予言**: 「不況が来る」と皆が思えば実際に不況が来る
- **ナラティブ経済学**: 「この国は成長している」という物語が投資を呼び、成長を生む

> 💡 **Tip**: *「経済変動の大部分は、アニマル・スピリッツの変動によって説明できる」* — アカロフ＆シラー

## 9. 外交・軍事システム

### 9.1 外交関係

- **二国間関係値**: -100（敵対）〜 +100（同盟）
- 貿易関係・イデオロギー的近さ・歴史的経緯で変動
- 外交アクション: 貿易協定、同盟締結、経済制裁、外交抗議、和平交渉

### 9.2 軍事ドクトリン

**兵制:**

| 選択肢 | コスト | 効果 | 政治的影響 |
|--------|-------|------|-----------|
| 徴兵制 | 安い | 大量だが低練度 | 極めて不人気 |
| 志願制 | 高い | 少数精鋭 | 「見えない戦争」が可能 |
| 傭兵/PMC | 非常に高い | 政治的コスト低 | accountability の欠如 |

**戦略姿勢:** 孤立主義 / 集団安全保障 / 覇権追求 / 核抑止

**ドクトリン:** 攻勢 vs 防勢 / 通常戦力 vs 非対称戦力 / 海軍 vs 陸軍 vs 空軍重視

> 💡 **Tip**: *「戦争とは、他の手段をもってする政治の延長にほかならない」* — クラウゼヴィッツ

### 9.3 戦争の処理 — 戦略的意思決定

#### 戦争目的の設定

限定戦争 / 全面戦争 / 懲罰戦争 — 目的が大きいほど継続コスト増。

#### 進軍先の選定

| 戦域 | 特徴 | リスク |
|------|------|-------|
| 敵の首都 | 早期終結を狙える | 兵站延伸。抵抗が最も激しい |
| 国境地帯 | 兵站容易。防衛しやすい | 決定打にならない |
| 資源地帯 | 経済基盤の破壊/奪取 | 施設破壊で価値低下 |
| 港湾・海峡 | 海上封鎖で経済的締め上げ | 海軍力が必要 |

#### 戦争中の決定（毎ターン）

| 決定項目 | 選択肢 |
|---------|--------|
| 攻勢継続 vs 停止 | 攻撃続行 / 一時停止 / 防御移行 |
| エスカレーション | 通常戦 / 総動員 / 焦土作戦 |
| 講和の提案 | 現状維持 / 領土要求 / 賠償要求 |
| 戦線の優先順位 | 戦力の再配分 |

#### 戦争結果の自動計算

```
戦域ごとの結果 = f(投入戦力, ドクトリン相性, 地形, 兵站, 士気, 天候, 乱数)
```

> 💡 **Tip**: *「素人は戦略を語り、玄人は兵站を語る」* — ブラッドレー元帥

#### 戦争の政治的コスト

- 人的損失 → 不満度上昇
- 軍事費増大 → 他の支出を圧迫
- 勝てば支持率上昇、負ければ体制の正当性喪失
- **行動経済学的効果**: サンクコストの誤謬（「撤退できない」）、楽観バイアス（「あと少しで勝てる」）、ラリー効果の減衰

## 10. 社会・人口システム

### 10.1 人口動態

- **人口転換モデル**: 多産多死→多産少死→少産少死→高齢化
- 出生率は一人当たりGDP・教育水準・都市化率・社会保障に依存
- 移民は経済機会・政治的自由・治安で決まる
- **人口ボーナス/オーナス**: 生産年齢人口比率が経済成長に影響

### 10.2 教育

- 初等→中等→高等教育の段階的投資
- 効果は長期的（10〜20ターン）
- 教育水準は技術革新能力・官僚の質・民主主義の成熟度に影響

### 10.3 階層と格差

- ジニ係数の動的変化
- 格差大: 社会不安↑、消費↓ / 格差小すぎ: 成長インセンティブ↓

> 💡 **Tip**: *「r > g が成り立つ限り、格差は自然に拡大する」* — トマ・ピケティ

### 10.4 技術と革新

- 研究開発投資 → 全要素生産性 A の上昇
- 後発国のキャッチアップ（条件: 教育水準・開放性）
- **技術革命の前提条件と効果**:

| 技術革命 | 前提条件 | 制度的効果 |
|---------|---------|-----------|
| 農業革命 | 定住・適切な気候 | 余剰→分業→階層社会→国家 |
| 商業革命 | 余剰農産物・交易路の安全 | 商人階級→代議制の萌芽→資本主義の基盤 |
| 産業革命 | 財産権・資本蓄積・識字率・エネルギー資源 | マルサスの罠脱出→都市化→民主化 |
| 情報革命 | 高度な教育・資本市場・知的財産権 | プラットフォーム独占→監視技術→労働市場変化 |

> 💡 **Tip**: *「創造的破壊こそが資本主義の本質的事実である」* — シュンペーター

## 11. ランダムイベント

**自然災害:** 地震、洪水、干ばつ、疫病 → インフラ破壊、人口減少
**政治イベント:** カリスマ的指導者の登場、スキャンダル、暗殺
**経済イベント:** 資源発見、交易路変化、技術的ブレークスルー
**外交イベント:** 隣国の革命、大国の介入、宗教対立

危機を利用した改革（ショック・ドクトリン）: 平時にはできない改革が危機後なら可能に。

> 💡 **Tip**: *「深刻な危機を無駄にしてはならない」* — ラーム・エマニュエル

## 12. ゲームの核心的ジレンマ集

| ジレンマ | 短期的に有利 | 長期的に有利 | 公共選択論的障害 | 行動経済学的障害 |
|---------|------------|------------|----------------|----------------|
| バターか大砲か | 社会保障充実 | 適度な国防費 | 軍部 vs 福祉ロビー | 脅威の過小評価 |
| 自由貿易か保護主義か | 保護 | 自由貿易 | 損失は集中し利益は拡散 | 損失回避 |
| 財政赤字 | 借金で人気取り | 均衡財政 | 将来世代は投票不能 | 双曲割引 |
| 改革か現状維持か | 現状維持 | 改革 | 既得権益のロックイン | 現状維持バイアス |
| 中央銀行の独立性 | 政治的コントロール | 独立性維持 | 政治家は金融緩和を望む | 現在バイアス |
| 教育投資 | 削減 | 増額 | 効果が10年後 | 計画の誤謬 |
| 腐敗対策 | 見て見ぬふり | 厳格な取り締まり | 腐敗した者が対策を決定 | 正常性バイアス |

## 13. UIとフィードバック

### 13.1 メイン画面

- **世界地図**: 国境・勢力圏・貿易路・戦域の可視化
- **国内パネル**: 主要経済指標
- **利益集団マップ**: バブルチャートで影響力と要求を表示
- **官僚パネル**: 各組織の予算・命題健全度・効率
- **制度パネル**: 現在の全制度一覧。解禁条件の充足度をプログレスバー表示
- **制度相性マップ**: 制度パッケージ全体の適合度を可視化
- **歪みモニター**: 強制導入した制度の歪み推移グラフ
- **歴史グラフ**: GDP・人口・腐敗度の時系列
- **ニュースティッカー**: ゲーム内イベントを新聞風に報道

### 13.2 ターン進行

```
1. 状況確認フェイズ
   └ 経済指標・世論・外交・官僚からの予算要求を確認

2. 政策決定フェイズ
   ├ 財政政策（税率・歳出配分・官僚予算の承認/削減）
   ├ 金融政策（金利・貨幣供給・中央銀行への指示）
   ├ 行動政策（ナッジの導入・制度設計）
   ├ 制度改革（自然導入/強制導入。サボタージュリスクあり）
   ├ 外交アクション
   └ 軍事方針（戦時: 戦域選択・進軍先・戦力配分）

3. 利益集団・官僚の反応フェイズ（自動）
   ├ 各利益集団のロビー活動
   └ 官僚の反応（サボタージュ/ワシントン記念碑作戦）

4. 解決フェイズ（自動）
   ├ NKモデル計算（GDPギャップ、インフレ、失業）
   ├ 行動経済学バイアスの適用
   ├ 官僚の自律行動
   ├ 制度の歪み変動
   ├ 世論変動・ランダムイベント
   ├ 外交・軍事処理
   └ 選挙（該当ターン）

5. 結果報告フェイズ
   └ 年次報告書。Tips表示。次ターンへ
```

### 13.3 Tips表示システム

ゲーム状況に応じて経済学者の名言・理論解説がポップアップ表示される。

**表示トリガー例:**
- インフレ急上昇 → フリードマンの引用
- 利益集団の影響力が閾値超過 → オルソンの引用
- 官僚予算が急膨張 → ニスカネンの引用
- バブル崩壊 → ミンスキーの引用
- 選挙前のバラマキ → ノードハウスの引用
- ゼロ金利到達 → クルーグマンの引用
- ジニ係数急上昇 → ピケティの引用
- 制度強制導入 → マキャヴェッリの引用
- 制度間不適合発生 → 青木昌彦の引用
- 戦争の泥沼化 → クラウゼヴィッツの引用

### 13.4 情報の階層化

- **表面**: GDP成長率、失業率、支持率（常時表示）

## 14. 学びとプレイの接点

このゲームを通じてプレイヤーが自然に学ぶこと：

- **なぜ「正しい政策」が実行されないのか** → 公共選択論
- **なぜ国家は衰退するのか** → 制度の硬直化、レントシーキング、官僚の病理
- **経済政策にはトレードオフがある** → フィリップス曲線、不可能の三角形
- **制度が経済成長を決める** → アセモグルの洞察
- **制度には相性がある** → 青木昌彦の制度的補完性
- **制度は強制できるが、歪みが生じる** → ガーシェンクロン、ノース
- **改革には正しい順序がある** → フクヤマの三要素
- **短期と長期の対立** → 割引率と政治的時間軸のミスマッチ
- **人間は合理的ではない** → 行動経済学。バイアスが政策を歪める
- **期待が経済を動かす** → ニューケインジアンの核心
- **官僚は味方ではない** → 組織は放置すれば自己増殖する
- **戦争は政治の延長である** → 軍事的勝利と政治的勝利は別物
- **歴史は制度の産物** → 転換点での選択が数世紀の経路を決定する

> 💡 **最後のTip**: *「経済学を学ぶ目的は、経済学の問いに対する答えを得ることではなく、経済学者に騙されないようにすることである」* — ジョーン・ロビンソン
